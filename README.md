#  ML Expense Analyzer

##  Project Description
This project predicts user financial behavior using Machine Learning techniques. It analyzes user data and provides insights into spending patterns.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

##  Features
- Data preprocessing
- Model training
- Prediction system
- Data analysis using CSV & Excel

##  How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn
2. Run the application:
   python app.py

##  Project Structure
- app.py → Main application
- data.csv → Dataset
- Bank_Personal_Loan_Modelling.xlsx → Additional dataset
- sample_data/ → Sample files
